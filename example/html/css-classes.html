CSS Classes Example - Color Flow Extension

Important: The Color Flow extension can now highlight CSS classes when colorFlow.enableClassHighlighting is set to true in settings. By default, this feature is false.

Default: Only style="..." inline attributes are highlighted.

When enabled: Both inline styles AND CSS classes defined in <style> tags are highlighted.

Understanding Color Flow Extension

Color Flow Extension Scope:

  • ‚úÖ Always: Highlights inline styles: style="color: red;"
  • ‚ùå Never: Highlights external CSS files
  • üî≤ Optional: When colorFlow.enableClassHighlighting = true, highlights CSS classes in <style> tags: class="text-red"

Default Behavior: The extension parses HTML documents and only looks at inline style attributes.

With Class Highlighting Enabled: When colorFlow.enableClassHighlighting is set to true, the extension also parses CSS from <style> tags and applies highlights to elements using those classes.

Use Case: Best for quick prototyping, debugging inline styles, and visualizing color in HTML documents. Enable class highlighting to also see colors from embedded CSS.
